Hypothesis Testing

Type I error (α) and Type II error (β) are in tension — reducing one raises the other. Power = 1 − β is what good tests maximise.

GATE Angle

p-value ≠ P(H₀ is true). It's P(data this extreme | H₀ true). A very common confusion.

α ↓ ⟹ power ↓ (harder to detect real effect). Increasing n raises power without changing α.

Two-tailed test: Reject if |z| > z_α/2 (e.g., 1.96 for α=0.05). p-value = 2×P(Z≥|z_obs|).

t-test vs z-test: Use t when σ unknown (use sample s). t → z as n → ∞.
