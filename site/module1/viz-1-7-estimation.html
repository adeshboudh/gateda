Estimation & Confidence Intervals

A confidence interval either contains the true parameter or it doesn't. The "95% confidence" refers to the procedure — 95% of CIs constructed this way will contain the true value.

Visualization 2 — CI Width vs Sample Size

CI width shrinks as 1/√n — doubling precision requires 4× more data.

Width formula

GATE Angle

To halve CI width, quadruple n (from n to 4n). Extremely common GATE calculation.

MLE properties: Consistent (converges to true θ), asymptotically normal, asymptotically efficient (achieves Cramér-Rao bound).

Width depends on: z (confidence level), σ (population spread), n (sample size). It does NOT depend on population size N (unless n/N > 0.05 — finite population correction).
