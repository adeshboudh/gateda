GATE Angle

E[X²] ≠ (E[X])². Always: E[X²] = Var(X) + (E[X])². This is the most common trap.

Independence: E[XY] = E[X]·E[Y] only if X, Y independent. Var(X+Y) = Var(X)+Var(Y) only if independent.

Var(aX+b) = a²Var(X): b shifts, doesn't spread. a² not a — very commonly tested.
