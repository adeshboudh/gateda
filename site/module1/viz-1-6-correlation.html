GATE Angle

Key formula: Cov(X,Y) = E[XY] − E[X]E[Y]. For independent X,Y: Cov = 0 (but NOT vice versa — see nonlinear preset!).

Linearity: Cov(aX+b, cY+d) = ac·Cov(X,Y). Var(X+Y) = Var(X) + Var(Y) + 2·Cov(X,Y).

r range: −1 ≤ r ≤ 1. r = ±1 iff perfect linear relationship. r = 0 does NOT mean independent.
